# HTML/CSS Portfolio

This is a static HTML/CSS portfolio website that you can host on any server including XAMPP.

## Files Included

- `index.html` - Main landing page
- `styles.css` - All styling
- `biodata.html` - Personal biodata page
- `recruiters.html` - For recruiters page
- `clients.html` - For clients page
- `peers.html` - For peers page
- `collaborators.html` - For collaborators page

## How to Use

1. Copy all files to your web server directory (for XAMPP: `htdocs` folder)
2. Access the website through your browser:
   - For XAMPP: `http://localhost/html-portfolio/`
   - Or your server URL

## Customization

- Edit the HTML files to update content
- Modify `styles.css` to change colors and styling
- Add your own background image by replacing the hero-background URL
- Update meta tags with your actual information
